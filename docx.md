## Краткая информация

*Данный проект является плагином для Intellij IDEA*


### Структура
проект представляет реализацию шаблона IDE plugin в Intellij IDEA с языком java и сборщиком gradle и имеет единственную реализацию действия реагирующего
на вызов event и при обнаружении нажатия клавиш ctrl + alt + / закрывает текущий файл
если ни какой файл не открыт, то ничего не происходит

#### actionPerformed

есть класс MyPlugin
от наследованный от AnAction

внутри него есть метод
метод проверяющая горячие клавиши и закрывающие текущий файл
void actionPerformed(AnActionEvent event)

event - очередное событие, который поймал жизненный цикл
