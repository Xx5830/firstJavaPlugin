## Краткая информация

*Данный проект является плагином для Intellij IDEA*


### Структура
проект представляет реализацию шаблона IDE plugin в Intellij IDEA с языком java и сборщиком gradle и имеет единственную реализацию действия реагирующего
на вызов event и при обнаружении нажатия клавиш ctrl + alt + / закрывает текущий файл
если ни какой файл не открыт, то ничего не происходит

## MyPlugin
Основной рабочий класс плагина.
Наследуется от класса AnAction, паредоставляемым API для разработки плагинов

### actionPerformed
метод **actionPerformed**\(AnActionEvent event\)

***Принимает:**

event (AnActionEvent) - очердное событие*

**описание:**

метод класса, проверяющая горячую клавишу и закрывающия текущий файл
при его обнаружении
